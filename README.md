# 项目简介

## 核心功能
- 基于YOLOv8的钢缆缺陷检测系统，可识别'break'和'thunderbolt'两类缺陷
- 支持图片、视频及摄像头进行检测
- 支持批量检测并在界面直接查看所有检测结果
- 支持检测结果保存
- 支持选择不同检测模型
- 支持自定义置信度和IOU阈值
- 支持选择是否显示标签和原图
- 界面实时显示目标位置、目标总数、置信度等信息

## 适用场景
- 钢缆缺陷检测
- 工业设备安全检查
- 视觉识别分析

## 技术特点
- 使用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 技术栈包括Python3.9、opencv-python、PyQt5、ultralytics、torch1.9等
- 提供完整源码、标注数据集(1318张钢缆缺陷图片)、训练好的模型及训练结果
- 配置成功环境可直接使用，附带详细运行教程
